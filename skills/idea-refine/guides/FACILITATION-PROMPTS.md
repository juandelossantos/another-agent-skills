# Facilitation Prompts for Idea Refinement

> **Sources:** IDEO Design Kit — facilitated ideation methods. d.school bootleg bootcamp — brainstorming rules and question design. Inversa — Assault on the Premise for first-principles questioning.

## Opening / Problem Framing

Use to set the stage before diverging:

- "How might we [outcome] for [user] without [constraint]?"
- "Can you name 3 specific people who have this problem right now?"
- "What are they doing today instead of using a solution?"
- "Is this a painkiller or a vitamin? Painkiller signs: emotional description, workarounds exist, willingness to pay."

## Divergent Phase Prompts

Use to push beyond the obvious:

- "What if we did the opposite?"
- "What if there were no top-level navigation?"
- "What if budget/time/tech weren't factors?"
- "What if this were for a completely different user?"
- "What would this look like at 10x the scale?"
- "What if it could only have one feature?"
- "What would [domain expert] find obvious that outsiders miss?"
- "What are we assuming that feels obvious but isn't validated?"

## Convergent Phase Prompts

Use to filter and stress-test:

- "What could kill this idea? It's 12 months from now and it failed — list every reason."
- "What must be true for this to work? (Dealbreakers first.)"
- "Is this differentiation durable? Can a competitor copy it in a week?"
- "Would users switch from their current approach? What would make them switch?"
- "If you can't name one thing this does that nothing else does, that's a problem."
- "Which failure modes are acceptable? Which would kill the project?"

## Commitment Prompts

Use before moving to spec:

- "Is the selected direction specific enough to spec? Or does it need another refinement cycle?"
- "What are we choosing NOT to do by picking this direction?"
- "Does anyone disagree with this direction? I need to hear the objection, not just silence."
- "On a scale of 1-10, how confident are we? If under 7, what would get us to 7?"
