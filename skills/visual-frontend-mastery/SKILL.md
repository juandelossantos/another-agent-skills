# Visual Frontend Mastery

**Distinctive, high-performance frontend interfaces.**

This skill covers the visual and creative aspects of frontend development: intentional visual design, high-performance animations, and distinctive UI. It references platform skills for implementation.

## Relationship

```
engineering-fundamentals (foundation philosophy)
└── frontend-ui-engineering (universal UI principles)
    └── visual-frontend-mastery (this skill — visual direction + animations)
        └── frontend-web (web implementation)
        └── frontend-mobile (mobile implementation)
```

## When to Use

- Creating visually distinctive landing pages or dashboards
- Adding high-performance animations
- Making design decisions about look and feel
- Building React/Next.js components with visual polish

## When NOT to Use

- Platform-agnostic UI principles (use frontend-ui-engineering)
- Web-specific implementation details (use frontend-web)
- Backend or API work
