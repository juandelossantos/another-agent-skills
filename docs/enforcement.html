Enforcement

Mechanical enforcement, not suggestions.

Level 1: Process Rules

Agent remembers rules. Load-bearing for simple tasks.

Level 2: Visible Manifest

Agent presents DECISION POINT before every mutation.

Level 3: SHA256 Hash-Bound Token

Hook verifies integrity. Agent cannot bypass what it cannot forge.

Level 4: Manifest Gate

Agent cannot generate tokens without writing a commit manifest first.

Honest Limitations

⚠

These gates create friction, not guarantees. The human must stay alert.
